Monaco Editor 内核。语法高亮、智能补全、代码折叠、括号配对着色、Ctrl + 滚轮缩放。官方 IDE 简陋编辑器的全面升级。

⚡

一键编译 & 上传

基于官方 Arduino CLI，支持流式编译反馈 + 实时进度条。编译错误精确到行列，点击即可跳转。

串口监视器 & 绘图仪

300 ~ 921600 波特率全支持，实时数据收发。内置绘图仪，传感器波形可视化。官方 2.0 才有绘图仪，我们 0.1 就有。

☷

板卡 & 库管理

Uno / Mega / Nano / MKR / Portenta / ESP32 / STM32 / RP2040 全系列支持。在线搜索安装库，一键完成。

☸

灵活的面板布局

侧边栏 / AI 面板 / 底部输出面板 — 三处均可拖拽调节宽度和高度。不像官方 IDE 那样只能固定布局。

安装后自动检测缺少的平台，弹窗引导一键安装 AVR 包。官方 IDE 需要自己去 Boards Manager 里找。新手友好。
